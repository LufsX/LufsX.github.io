介绍

Baidu-PCS-Go是一款支持PC/移动设备的仿 Linux shell 文件处理命令的百度网盘命令行客户端.

准备

需要的东西

  • 一部手机
  • 一定的基础知识
  • Neoterm | 下载链接
    理论上其他终端也行，只是我觉得Neoterm好看一点

如果准备好了
那么
开始游戏吧
开始吧

教程

Neoterm端准备工作

下载并安装Neoterm
开始
等待进度条跑完，会有点慢，慢慢等

然后需要下载wget和zip，具体方法是
执行apt install zip wget -y

获取最新版BaiduPCS-Go

打开官方开源地址 https://github.com/iikira/BaiduPCS-Go/releases
找到你手机的对应版本，例如我的安卓机为arm64，那就选择BaiduPCS-Go-v3.5.6-android-21-arm64.zip

Neoterm终端执行

wget -O 1.zip https://github.com/iikira/BaiduPCS-Go/releases/download/v3.5.6/BaiduPCS-Go-v3.5.6-android-21-arm64.zip

意思是下载https://github.com/iikira/BaiduPCS-Go/releases/download/v3.5.6/BaiduPCS-Go-v3.5.6-android-21-arm64.zip的文件并重命名为1.zip

接着执行

unzip 1.zip && rm 1.zip

意思是解压1.zip并删除1.zip

需要进行一波重命名操作
执行

mv BaiduPCS-Go-v3.5.6-android-21-arm64 baidu

注意，BaiduPCS-Go-v3.5.6-android-21-arm64需要根据你下载的文件名字而定
例如下载的是BaiduPCS-Go-v3.5.6-android-21-arm64.zip
那么就是BaiduPCS-Go-v3.5.6-android-21-arm64
此项意思为重命名解压文件夹为baidu

接着，执行

cd ~/baidu && ./BaiduPCS-Go

即可运行起来

必要的设置&过程

既然运行起来了那肯定是要登录咯
执行login即可
输入账号密码即可
注意：输入密码时密码是不可见状态
注意：如需输入验证码请复制验证码链接，在浏览器打开
注意：如登录失败，再次输入login即可重新登录

登录后，执行

config set --appid 265486

执行后关闭Neoterm或执行exit
接着在一个登录了此账号的浏览器打开此链接

然后在度盘里的我的应用数据下就会出现baidu_shurufa的文件夹
如果没有，请自行创建

一些设置（必看！！！）

设置下载文件的保存位置
例如下载的文件保存到/sdcard/Download/BaiduNetdisk下
执行

config set --savedir '/sdcard/Download/BaiduNetdisk/'

设置下载线程，官方默认100线程，可以设置50～500
例如设置400下载线程
执行

config set --max_parallel 400

设置同时下载线程
例如设置为5
执行

config set --max_download_load 5

使用（必看！！！）

进入终端输入cd /baidu/ && ./BaiduPCS-Go即可运行
将想下载的文件移动到/我的应用数据/baidu_shurufa/
接着执行cd /app/baidu_shurufa/
然后输入d 将下载的文件名字即可下载

一些命令&操作技巧
help 帮助
ls 可列出当前文件和文件夹
cd 路径 可进入某路径
d 你要下载的文件 下载某文件
locate 你要获取直链的文件 获取某文件直链
logout 退出登录
停止下载按住ctrl并输入c即可
多任务下载直接多开窗口即可
